# Шаг 10. Финальная интеграция

- версия MDVWB 1.2.0;
- `mdvwb-manager.service` управляет произвольным количеством шин;
- существующие `/etc/default/mdvwb-N` мигрируют в `/etc/mdvwb/buses.json`;
- статическая страница устанавливается в `/mnt/data/www/mdvwb/`;
- драйвер сам публикует retained-метаданные `Fan-*` и `sist-*`;
- старый wb-rules-скрипт с `ArrID` автоматически отключается с сохранением резервной копии;
- удалённые из конфигурации устройства очищаются из retained MQTT;
- GitHub Actions собирает `MDVWB`, `mdvwb-manager`, страницу и установщик в одном ARM64 artifact.
